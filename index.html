Blog Kegiatan Magang di Dago Engineering tahun 2024

February 2024

Langkah awal dimulai dengan merancang konsep dasar game yang akan dikembangkan. Kegiatan diawali dengan mengikuti workshop AIMS untuk mempelajari dasar-dasar pengembangan game, yang diikuti dengan pengunduhan Unity sebagai platform utama yang akan digunakan. Selain itu, saya juga membuat Trello untuk mengatur tugas-tugas proyek dan merancang konsep awal game. Pembuatan slide presentasi untuk memaparkan konsep game dilakukan untuk memastikan ide yang saya miliki dapat dipahami dengan baik oleh tim.

Setelah presentasi, fokus beralih ke revisi dan perbaikan konsep game berdasarkan masukan yang diterima. Dalam proses ini, saya juga mencari berbagai asset game yang relevan dan membuat preview mockup game untuk visualisasi awal. Finishing mockup di Figma menjadi langkah penting untuk mendapatkan gambaran yang lebih jelas tentang tampilan akhir game. Di pertengahan bulan, desain logo untuk Dago Game mulai dikerjakan. Bersamaan dengan itu, finalisasi mockup prototype di Figma dilakukan, diikuti dengan diskusi intensif dan presentasi final konsep game. Setelah mendapat persetujuan, perencanaan pengembangan game dimulai dengan menyusun timeline dan sprint untuk memastikan setiap tahap pengembangan berjalan sesuai jadwal. Persiapan tools dan akun untuk pengembangan menjadi langkah terakhir di bulan ini, mempersiapkan semuanya untuk memulai coding di bulan berikutnya.

March 2024

Maret dimulai dengan setup proyek Firebase dan Github untuk kolaborasi, diikuti dengan setup proyek Unity. Langkah ini memastikan bahwa semua anggota tim dapat berkolaborasi dengan baik dalam pengembangan game. Pembuatan repository di Github, design awal, dan fixing error di Unity menjadi fokus awal bulan. Kemudian, saya melanjutkan pengembangan game di Unity dan melakukan diskusi singkat untuk membahas progres yang telah dicapai.

Pada minggu kedua, kegiatan utama berfokus pada slicing splashscreen dan menu utama, mencoba connect Firebase, serta pengumpulan data untuk database. Pembuatan Keytore aplikasi dan scene game settings dilakukan, diikuti dengan debugging untuk memastikan semuanya berjalan lancar. Proses ini berlanjut dengan slicing awal board scene, import data ke Firestore, dan meeting dengan anggota Dago Engineering untuk diskusi projek. Di akhir bulan, identifikasi hint di game board, pembuatan script query dari Firestore, dan improvement UI menjadi prioritas. Setelah berhasil retrieve data dari Firestore, saya mencari referensi untuk query data dan melakukan testing Crashlytics Firebase untuk memastikan stabilitas aplikasi.

April 2024

April dimulai dengan penyelesaian pembuatan script untuk check jawaban dan logika game. Selain itu, saya mengembangkan script untuk menentukan pemenang dan menyelesaikan kondisi menang untuk player. Improvisasi UI juga dilakukan untuk meningkatkan pengalaman pengguna. Menambahkan opsi surrender dan offer draw menjadi fitur baru yang ditambahkan, serta membuat board dinamis yang dapat berganti acak otomatis.

Di minggu kedua, saya menambahkan waktu di game, mengoptimasi database, dan mengaktifkan Google Authentication. Selain itu, scene tutorial dan scene awal single player mulai dibuat. Fokus beralih ke improvisasi UI dan pengembangan single player mode, di mana saya meningkatkan game logic dan memperbaiki bug yang ditemukan. Pada minggu ketiga, design algoritma single player mulai dikerjakan, dan peningkatan beberapa UI dilakukan. Sorting database dan penambahan data ke database menjadi langkah penting untuk memastikan data game terorganisir dengan baik. Fokus utama adalah finishing game board untuk single player dan fixing bug.

Bulan ini diakhiri dengan diskusi progress, improvement UI, dan pengembangan single player mode lebih lanjut. Penambahan interaksi UI/UX di single player dan finishing game logic untuk single player menjadi prioritas, diikuti dengan improvisasi logic dari versus mode.

May 2024

Bulan Mei dimulai dengan finalisasi algoritma single player dan pembuatan scoring single player. Saya melakukan diskusi progres game, inisiasi Firebase Auth, serta penyusunan laporan dan teknis game. Konstruksi database baru untuk revisi design ulang interface menjadi langkah penting untuk mengoptimalkan kinerja game. Pada minggu kedua, migrasi database Firestore dan konversi ke JSON untuk database lokal menjadi fokus utama. Saya mengubah script game Unity dari Firestore ke JSON lokal dan menyelesaikan game untuk building APK. Testing bug dan gameplay dilakukan secara intensif, diakhiri dengan publikasi game ke Play Store.

Minggu ketiga diisi dengan perancangan fitur-fitur untuk game Ojol, pembuatan presentasi proyek game Ojol, dan pengembangan presentasi serta fitur game Ojol Adventures. Saya juga belajar dasar engine Unity 3D dan mengadakan meeting untuk membahas rencana game Ojol.

June 2024

Pada bulan Juni, pengembangan tampilan main menu menjadi fokus utama. Saya menambahkan tampilan 3D dinamis saat pemilihan item di main menu, serta placeholder untuk atribut stats player. Placeholder stats bar untuk item equipment juga ditambahkan di main menu, bersama dengan variabel untuk menyimpan data uang di app data. Penambahan placeholder minimap dan GPS, serta pengujian algoritma path finding dilakukan di minggu kedua. Saya juga menambahkan bangunan di map dan menguji gameplay.

Pada minggu ketiga, penambahan button dan script untuk memilih kendaraan di main menu dilakukan, diikuti dengan penambahan icon dan animasi simulasi hujan. Bulan ini diakhiri dengan diskusi progress game dan penambahan loading screen dari menu ke game core. Bug fixing dan optimasi script pemilihan motor juga dilakukan, serta pembuatan akun Google untuk pre-testing aplikasi. Monitoring dan evaluasi bersama pak Wahyu dan mentor menjadi langkah penting untuk memastikan kualitas game sebelum di test oleh mentor magang.

July 2024

Bulan Juli diawali dengan pembaruan codingan pemilihan motor untuk di open world dan kunjungan serta presentasi hasil magang kepada WD1 (Pak Windu). Weekly meeting dengan staff Dago Engineering diadakan untuk membahas progres lebih lanjut. Pada minggu kedua, saya mengadakan meeting dan presentasi progress game kepada Direktur Utama PT Dago Engineering, yang menjadi penutup dari rangkaian kegiatan magang ini. Semoga semua usaha yang dilakukan dapat menghasilkan game yang berkualitas dan bermanfaat bagi para pengguna.
